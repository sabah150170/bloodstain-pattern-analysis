# bloodstain-pattern-analysis
mage Processing-Based Determination of the Angle of Origin in Bloodstain Pattern Analysis (Görüntü İşleme İle Kan Lekesi Model Analizinde Darbe Noktası Tespiti) 




## Dataset

The images used in this study were obtained from the `experiments_with_guns` directory of the [BPA Scans dataset](https://www.kaggle.com/datasets/marcoferrarini/bpa-scans) available on Kaggle. These images were specifically used to prepare the dataset and train the machine learning model developed for bloodstain detection and validation.
