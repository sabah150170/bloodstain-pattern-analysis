# Bloodstain Pattern Analysis Project

This repository contains Python software, machine-learning scripts, generated CSV annotation data, and a trained ellipse-confidence model.

## Dataset Source

The CSV data were generated with custom scripts by processing images from the `experiments_with_guns` directory of the [BPA Scans dataset](https://www.kaggle.com/datasets/marcoferrarini/bpa-scans).

No original, cropped, modified, or annotated BPA Scans images are included in this repository.
