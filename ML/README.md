# Machine-Learning Pipeline

- `prepare_manual_labelling.py`: prepares ellipse candidates.
- `manual_labeler.py`: assigns validation labels.
- `training.py`: trains the ellipse-confidence model.
- `label_data/`: contains generated CSV files.
