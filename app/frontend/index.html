User Guide

User Guide

This application provides an interactive workflow for bloodstain pattern analysis and Area of Origin (AoO) reconstruction.

1. Upload Image

Use the Upload Image section to load the image that will be analyzed.

2. Define Surface Coordinates

Using the coordinate system, enter the real-world coordinates of the image corners in centimeters (cm). These values represent the physical dimensions of the surface on which the bloodstains are located.

3. Select Validation Mode

Choose how detected ellipses are validated before analysis.

  • ML-Based Validation:
    Ellipses are validated through a machine learning–based filter. No user action is required during this process.
  • Rule-Based Validation:
    Ellipse validation is performed using predefined rules and threshold values.

4. Analysis Results

After completion, the analysis report is displayed in the side panel. The report includes the results of two different Area of Origin (AoO) reconstruction algorithms along with the calculated coordinates of the impact source or sources.

5. 3D Visualization

A three-dimensional visualization of the scene is generated in the main view. This visualization represents the 3D reconstruction of the coordinates reported in the analysis.

Using the sidebar, the visualization can be switched between Tangent and Least Squares methods. By default, the Tangent method visualization is displayed.
