# Ellipse Confidence Model

File: `ellipse_confidence_model_manual.pkl`

The model was trained with CSV features and labels generated by this project's scripts using BPA Scans images. No source images are included.

Intended use: academic and research use in experimental bloodstain pattern analysis.

License: `CC BY-NC 4.0`.

The model must not be used as the sole basis for forensic or legal decisions.
